# JoymonOnline.In
Source code of my personal site www.joymononline.in.

# How to run
- Clone
- npm install
- build - npm run devBuild
- Serve - npm run start

# Deployment

## Development
-   Automatically after each each from AppVeyor

## Production
- Create a release.
- Release create a tag and Travis CI will release when there is a tag.
